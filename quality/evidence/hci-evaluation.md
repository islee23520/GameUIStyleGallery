# HCI Evaluation Evidence

HCI evidence helps separate task success, perceived usability, aesthetics, and cognitive load.

## Evidence Types

- heuristic evaluation;
- cognitive walkthrough;
- persona or task review;
- first-impression or perceived-aesthetics evidence;
- user testing or validated instruments when required.

## Can Support

- a task path has been reviewed;
- a usability heuristic has an observed issue or pass condition;
- a design claim needs human review rather than static validation;
- empirical evidence is required before making a strong user-perception claim.

## Cannot Prove

- universal user preference;
- aesthetic quality from a single expert opinion;
- task success without a task scenario;
- user perception without user or validated-instrument evidence.
